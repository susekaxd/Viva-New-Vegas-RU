Установка

В этом разделе, мы выполним чистую установку и настройку игры, необходимую для руководства.

Если вы столкнулись с проблемами в процессе выполнения инструкций, проверьте, нет ли решения в FAQ.
Если ваша проблема не описана здесь, присоединяйтесь к Discord (EN) для помощи.

Место установки

Чистая установка требуется не только для того, чтобы избавиться от следов установленных ранее модификаций, но и для того чтобы игра была установлена в безопасном месте. Вещи на которые нужно ориентироваться при выборе места установки:
  • Избегайте любых системных папок Windows: В частности, к ним относятся C:\Program Files, C:\Program Files x86, а также папки Рабочий стол и Документы.
    Это связано с тем, что пользователи не имеют доступа на запись в Program Files, в то время как Рабочий стол и Документы часто управляются OneDrive. Это может вызвать множество проблем с игрой и любыми инструментами для моддинга.
    Пример безопасного места установки: C:\Games
  • По возможности установка на SSD: Это значительно улучшит время загрузки и уменьшит статтеры.

Удаление игры

Если вы никогда прежде не устанавливали игру, пропустите эту часть.
Так как игра доступна в Steam, GOG и Epic Games, здесь представлены отдельные инструкции для каждой версии.
Steam
Показать
Если вы уже удалили игру, следуйте пунктам 4 и 5, чтобы убедиться, что не осталось старых файлов.
  1. Откройте Steam и зайдите в Библиотеку.
  2. Найдите Fallout: New Vegas в списке игр.
  3. Нажмите правой кнопкой мыши по ней, выберите Управление -> Удалить с устройства.
  4. Найдите папку Steam\steamapps\common и, если она есть, удалите папку Fallout New Vegas (Fallout New Vegas enplczru для региональной версии).
  5. Затем найдите папку Документы\My Games\FalloutNV и удалите все INI файлы в ней.
GOG
Show
  1. Open GOG and go to your Library.
  2. Find Fallout: New Vegas in the list.
  3. Right-click on it and select Manage Installation -> Uninstall.
    • If you used an offline installer then run the unins000.exe file in the game's Root folder.
  4. Navigate to Documents\My Games\FalloutNV and delete all INI files inside it.
Epic Games
Show
If you already uninstalled the game, make sure to follow step 4 to ensure there are no leftovers.
  1. Open the Epic Games Launcher and go to your Library.
  2. Find Fallout: New Vegas in the list.
  3. Click on 3 dots and select uninstall from the menu.
  4. Navigate to Documents\My Games\FalloutNV_Epic and delete all INI files inside it.



Установка игры

Steam
Show
Строго рекомендуется устанавливать игру вне системных папок Windows (таких как Program Files (x86)), поскольку строгая защита этих папок в Windows может привести к поломке некоторых модификаций/инструментов. Вместо этого мы воспользуемся инструментом Steam Library Setup Tool, чтобы создать новую папку библиотеки Steam вне папок Windows.

Если у вас уже есть Steam, установленный вне системных папок, следуйте 7 пункту и установите игру в свою библиотеку.
  1. Полностью закройте Steam, используя Диспетчер Задач или системный трей.
  2. Скачайте steam_library_setup_tool-3.2.exe по ссылке .
  3. Запустите файл и добавьте новую строку, кликнув на Add Row.
  4. Выберите путь для создания библиотеки, например C:\Games\Steam
  5. Нажмите Accept, а затем Yes, если появится окно о создании новой папки.
  6. Программа предложит выйти, нажмите OK.
  7. Откройте Steam и зайдите в Библиотеку.
  8. Найдите Fallout: New Vegas в списке и нажмите Установить.
  9. Под надписью Установить на:, выберите папку, созданную программой: второй пункт C:\, если вы устанавливаете на основной диск.
Если вы не увидели свою новую папку, перезагрузите компьютер.
  1. Дождитесь конца установки.
GOG
Show
  1. Open GOG and go to your Library.
  2. Find Fallout: New Vegas in the list and select Install.
  3. Set the Install to location to a location of your choice.
    • It is required to install the game outside of any default Windows folders (e.g., Program Files x86).
Epic Games
Show
  1. Open the Epic Games Launcher and go to your Library.
  2. Find Fallout: New Vegas in the list.
  3. Click on the game to install it.
    • It is required to install the game outside of any default Windows folders (e.g., Program Files x86).



После установки

Термины

Теперь, когда игра установлена, есть две папки, на которые мы будем часто ссылаться в этом руководстве:
  • Корневая папка: там, куда установлена игра.
    • Для Steam: Steam\steamapps\common\Fallout New Vegas (Steam\steamapps\common\Fallout New Vegas enplczru для региональной версии).
    • For GOG: GOG Galaxy\Games\Fallout New Vegas (unless you manually changed the installation path).
    • For Epic Games: Epic Games\FalloutNewVegas\Fallout New Vegas English (unless you manually changed the installation path).
  • Папка Data: Там, где хранятся ресурсы игры.
    • Для Steam: Steam\steamapps\common\Fallout New Vegas\Data (Steam\steamapps\common\Fallout New Vegas enplczru\Data для региональной версии).
    • For GOG: GOG Galaxy\Games\Fallout New Vegas\Data
    • For Epic Games: Epic Games\FalloutNewVegas\Fallout New Vegas English\Data

Отображение расширений файлов

По умолчанию, Проводник не будет показывать расширения файлов (такие как .exe, .dll или .esp). Расширения очень важны для следования руководству, поэтому очень рекомендуется включить их отображение:
  1. Open File Explorer. Откройте Проводник.
  2. Выберите панель Вид вверху.
  3. Поставьте галочку напротив Расширения имен файлов.

Создание новых INI файлов

  1. Запустите игру из Steam, GOG Galaxy, или через FalloutNVLauncher.exe.
  2. Нажмите OK в обоих всплывающих окнах Detecting Video Hardware.
    • Если всплывающих окон не появилось, найдите папку Документы\My Games\FalloutNV и удалите все INI файлы. Затем запустите игру снова.
  3. Выберите Options, затем нажмите на пресет Ultra.
    • Если у вас слабый ПК, или вы хотите добиться от игры максимальной производительности, вы можете выбрать пресет High. Он предлагает довольно схожее качество графики при приличном повышении производительности, особенно на слабых компьютерах. При желании можно установить еще более низкие настройки, но в этом случае производительность будет только уменьшаться. Еще один отличный способ повысить производительность без сильного ухудшения графики - отключить тени. В игре нет настоящих динамических теней, а только тени от NPC и врагов. Несмотря на незначительное улучшение графики, они могут сильно повлиять на производительность, особенно если в одной области находится много NPC.
  4. Выставьте в опции Resolution подходящее для вас разрешение.
  5. Нажмите OK, затем Exit.
